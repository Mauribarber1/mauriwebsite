# Mauri Barber Landing Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a bilingual (ES/EN) Next.js landing page for Mauri Barber, a barbershop in Barcelona, with hero, services, gallery, reviews, hours/contact and a WhatsApp booking flow.

**Architecture:** Next.js App Router, static content (no CMS/DB), a lightweight dictionary-based i18n pattern (no i18n library) mirroring the agency's existing `klentcreative` project: `app/[locale]/page.tsx` reads a JSON dictionary via `lib/dictionaries.ts` and passes typed props down to presentational components in `components/`. Tailwind v4 theme tokens (colors, fonts) are defined once in `app/globals.css`.

**Tech Stack:** Next.js (App Router) + TypeScript + React, Tailwind CSS v4, `next/font/google` (Playfair Display + Inter), `next/image`, Vercel deployment target.

**Testing approach note:** This is a static marketing page with no business logic to unit test. Instead of writing failing unit tests per step, each component task's "verify" step is a TypeScript typecheck (`npx tsc --noEmit`) after the file is created, and the final task does a full `next build` plus a visual check in the browser (dev server + preview tool). This is a deliberate adaptation of TDD for presentational-only code — every other principle (bite-sized steps, frequent commits, no placeholders) still applies.

- [ ] **Step 3: Write the dictionary loader**

`lib/dictionaries.ts`:
```typescript
import "server-only";
import esMessages from "@/messages/es.json";

const dictionaries = {
  es: () => import("@/messages/es.json").then((mod) => mod.default),
  en: () => import("@/messages/en.json").then((mod) => mod.default),
};

export type Locale = keyof typeof dictionaries;
export type Messages = typeof esMessages;

export async function getDictionary(locale: Locale): Promise<Messages> {
  return dictionaries[locale]();
}
```

```typescript
import { notFound } from "next/navigation";
import type { Locale } from "@/lib/dictionaries";

const LOCALES: Locale[] = ["es", "en"];

type Props = {
  children: React.ReactNode;
  params: Promise<{ locale: string }>;
};

export default async function LocaleLayout({ children, params }: Props) {
  const { locale } = await params;

  if (!LOCALES.includes(locale as Locale)) {
    notFound();
  }

  return <>{children}</>;
}
```

"use client";

import { useState } from "react";
import Link from "next/link";
import Logo from "@/components/Logo/Logo";
import WhatsAppButton from "@/components/WhatsAppButton/WhatsAppButton";
import type { Locale, Messages } from "@/lib/dictionaries";

type NavbarProps = {
  locale: Locale;
  navbar: Messages["navbar"];
  whatsappMessage: string;
};

export default function Navbar({ locale, navbar, whatsappMessage }: NavbarProps) {
  const [isMenuOpen, setIsMenuOpen] = useState(false);
  const otherLocale: Locale = locale === "es" ? "en" : "es";
  const otherLabel = locale === "es" ? "EN" : "ES";

  const links = [
    { href: "#services", label: navbar.services },
    { href: "#gallery", label: navbar.gallery },
    { href: "#reviews", label: navbar.reviews },
    { href: "#contact", label: navbar.contact },
  ];

  return (
    <header className="sticky top-0 z-40 border-b border-black/5 bg-paper/90 backdrop-blur">
      <div className="mx-auto flex max-w-6xl items-center justify-between px-6 py-4">
        <Link href={`/${locale}`}>
          <Logo />
        </Link>
        <nav className="hidden items-center gap-8 md:flex">
          {links.map((link) => (
            <a
              key={link.href}
              href={link.href}
              className="text-sm font-medium text-ink/80 transition-colors hover:text-gold-dark"
            >
              {link.label}
            </a>
          ))}
        </nav>
        <div className="flex items-center gap-4">
          <Link
            href={`/${otherLocale}`}
            className="hidden text-sm font-medium text-ink/60 transition-colors hover:text-gold-dark sm:inline"
